FOR FUN - O Festival de Carnaval do Landscape Pub

10/2/2007

Para quem não gosta de sambar no carnaval mas também detesta ficar parado. Para quem gosta "é de róque, poourra!". Para quem é doente da cabeça mas não do pé. Para quem também vai no Carnaval do Mal, ou para quem não vai. Para quem conhece o Landscape Pub e para quem ainda não conhece...

Vai rolar o FOR FUN, programação especial de carnaval do Landscape Pub. Cinco dias, cinco festas, que não são para todos os gostos mas atendem a uma boa série de bons e maus gostos. Se você não tem nenhum motivo melhor, vá só pela diversão. (há há há, entendeu o trocadilho terrível?)
